Quick Setup

Requirements

  • Windows 10/11 x64
  • No .NET install required (self-contained build)
  • ATS/ETS2 installed to use plugins
  • Game fatigue must be disabled
1

Extract

Unzip SimDuty-Release.zip to a folder of your choice (e.g., Documents/SimDuty).

2

Install Plugins

Copy the DLLs from the plugins folder to the game plugins folder:

  • ATS: American Truck Simulator/bin/win_x64/plugins
  • ETS2: Euro Truck Simulator 2/bin/win_x64/plugins
3

Drive

Launch SimDuty.exe first, then start your game. The system connects automatically when you enter the truck cabin.
