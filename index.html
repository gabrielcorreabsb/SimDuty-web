How to Install

Extract

Unzip SimDuty-Release.zip to a folder of your choice (e.g., Documents/SimDuty).

Install Plugins

Copy the DLLs from the plugins folder to the game plugins folder:

  • ATS: American Truck Simulator/bin/win_x64/plugins
  • ETS2: Euro Truck Simulator 2/bin/win_x64/plugins

Run

Open SimDuty.exe. The system will automatically detect the game when you enter the cabin.
